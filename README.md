# ScanWordToTranslate
一款基于Tesseract光学文字识别的扫词翻译词典
-
实现实时采集摄像机图层数据，并将数据转换成UIImage数据进行扫描，通过谷歌光学文字识别算法对UIImage处理，并将识别结果通过百度翻译API进行翻译，支持英译汉、汉译英、调用系统词典翻译。
因未对采集的UIImage进行区域裁剪，因而文字识别率很低，本实例仅用来提供解决方案，未做深入处理。
